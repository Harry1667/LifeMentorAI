# LifeMentorAI（圓桌智者）

AI 人生導師對話平台 — 多位 AI 智者角色同台，提供人生建議、決策輔助、心理支持。

## 功能
- 多角色 AI 導師選擇（圓桌智者陣容）
- 對話記憶與脈絡追蹤
- 使用者認證（Clerk）
- 對話歷史雲端儲存（Supabase）

## 技術棧
- Next.js（App Router）+ TypeScript
- Clerk（Auth）
- Supabase（Database）
- Anthropic API（Claude）

## 快速開始
```bash
cd 02-web
cp .env.example .env
npm install
npm run dev
```
